# Chittr-Chat-App

Front end app for Chittr server 
Made in React Native 

Designed in Lunacy (https://icons8.com/lunacy)
 
## Installation
 Needs android development environment(emulator, sdk, etc) and NodeJS

```bash
cd ChittChatApp
npm install 
```

## Usage

```python
npx react-native run-android

```
